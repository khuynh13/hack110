Project Submission

ALL APPS SHOULD RUN VIA AN HTML FILE!

Note: make sure that your script files are NOT named with "-app.ts" but are running out of an HTML file.
If you made a game and are not sure how to do this, see the below document as an example/guide (open in VSCode).~~Your app will struggle to publish otherwise.
If you are having trouble running your app, please come to office hours and we'll help you!

Everyone should also have a file called project-spec.html in their project folders that includes all information outlined under "Project Guidelines".
Both this file and your app HTML file should be published and linked separately in the submission form.
